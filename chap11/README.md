## 트랜잭션(TRANSACTION)

### 트랜잭션이란
<pre>
  더 이상 분할할 수 없는 최소 수행 단위, 한 개 이상의 데이터 조작 명령어(DML)로 이루어짐
  즉, 트랜잭션 내에 있는 여러 명령어를 한 번에 수행하여 작업을 완료하거나, 모두 취소 할수있음
</pre>
###### 트랜잭션을 제어할 때 사용하는 명령어를 TCL(Transaction Control Language)라고 한다.


<hr>

###### 예시

<pre>
  < 게시판 >
    
  -- 제목 클릭하면 상세페이지로 이동
  -- 제목, 내용, 작성자, 등록일자, 조회 등을 사용자에게 보여준다.

    SELECT TITLE, CONTENT, WRITER, REGDATE, COUNT 
    FROM BORAD
    WHERE NO = 1209  -- 1209번 게시물 제목을 클릭했다고 가정

    -- 제목 클릭시 상세페이지로 이동함과 동시에 조회수를 1씩 증가 시킴

    상세 페이지로 이동하는 도중에 통신장애가 발생했을 때
    1) 통신 장애와 상관없이 조회 수 1씩 증가(트랜잭션을 고려하지 않아도 됨)
    2) 통신 장애에 의해 사용자는 상세페이지를 보지 못했으므로 조회 수 1씩 증가 취소(트랜잭션 사용)
</pre>
